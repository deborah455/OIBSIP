# Password Generator (Python)

## Project Description
A Python application that generates secure random passwords based on a user-specified length.

## Features
- User chooses password length
- Includes uppercase and lowercase letters
- Includes numbers
- Includes special characters
- Generates strong random passwords

## Technologies Used
- Python 3
- random module
- string module

## How to Run

python3 app.py

## Example

Input:
Length = 12

Output:
Generated Password: aT9#kL2@pQ8!

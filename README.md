# Personal Productivity Assistant (Python)

## Project Description

A command-line personal assistant developed in Python that helps users perform common productivity tasks such as checking the time and date, searching for information, performing calculations, saving notes, and opening websites.

This project was developed as part of the Oasis Infobyte Python Programming Internship and demonstrates Python programming fundamentals, automation, file handling, and external library integration.

## Features

- Display current time
- Display current date
- Open Google in a web browser
- Search topics using Wikipedia
- Perform mathematical calculations
- Save notes to a text file
- Interactive command-based interface
- Error handling for invalid commands

## Technologies Used

- Python 3
- datetime
- webbrowser
- wikipedia

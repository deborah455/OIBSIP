# BMI Calculator (Python)

## Project Description
This is a simple Python program that calculates Body Mass Index (BMI) based on user input for weight and height. It also categorizes the BMI into standard health ranges.

## Formula Used
BMI = weight / (height^2)

## Features
- User input for weight and height
- BMI calculation
- Health category classification
- Clean formatted output

## How to Run
```bash
python3 app.py
